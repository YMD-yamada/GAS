# スプレッドシートのレイアウト

通知文は **1 枚目のシート** の次のセルから読み込みます。

| | A（ラベル・見やすさ用） | B（実際に送る値） |
|--|------------------------|-------------------|
| 1 行目 | 帰宅予定（ラベル） | 例：`19:30頃` |
| 2 行目 | 食事（ラベル） | 例：`あり` / `なし` |
| 3 行目 | メモ（ラベル） | 任意。空でも可 |

- **B1**：帰宅の目安  
- **B2**：食事の有無  
- **B3**：補足メモ（空なら送信文に「メモ：」行は出しません）

スクリプトを **スプレッドシートに紐づけて作成した**場合（「拡張機能 → Apps Script」から開いた場合）は、`SPREADSHEET_ID` は不要です。

**スタンドアロン**（[script.google.com](https://script.google.com/) だけで作ったプロジェクト）から別のシートを読むときは、スクリプトのプロパティに `SPREADSHEET_ID` を設定します。

## 運用の例

- 朝、B1〜B3 だけ編集する  
- 決まった時刻の時間トリガーで `sendFamilyUpdate` を実行する  
- または、送る直前に手動で `sendFamilyUpdate` を実行する
