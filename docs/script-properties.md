# スクリプトのプロパティ一覧

GAS エディタ左の **歯車アイコン → スクリプトのプロパティ → 行を追加** で、次の **プロパティ**（左列）と **値**（右列）を設定します。  
**値は GitHub に書かず、必ずここだけに保存してください。**

| プロパティ | 必須 | 説明 |
|------------|------|------|
| `LINE_CHANNEL_ACCESS_TOKEN` | はい | LINE Developers で発行した **チャネルアクセストークン（長期）** |
| `LINE_TO_ID` | はい | 通知を送る先 ID。個人宛ては **userId (`U`...)**、グループ宛ては **groupId (`C`...)** |
| `LINE_CHANNEL_SECRET` | いいえ | Webhook の **署名検証**に使う。未設定なら検証をスキップ（個人利用の簡易運用向け） |
| `SPREADSHEET_ID` | いいえ | 未使用（現行の Web アプリ版では使いません） |
| `FALLBACK_RETURN_TIME` | いいえ | 未使用（現行の Web アプリ版では使いません） |
| `FALLBACK_MEAL` | いいえ | 未使用（現行の Web アプリ版では使いません） |
| `FALLBACK_NOTE` | いいえ | 未使用（現行の Web アプリ版では使いません） |

`LINE_TO_ID` の自動保存は行わないため、誤送信防止のためにも手動で設定してください。

## スプレッドシート ID の見方

ブラウザでスプレッドシートを開いたときのアドレスが次の形です。

`https://docs.google.com/spreadsheets/d/【ここがID】/edit`

`【ここがID】` の部分を `SPREADSHEET_ID` に貼り付けます。

## トークンを載せてしまったら

1. LINE Developers で **チャネルアクセストークンを再発行**する。  
2. GAS のプロパティを新しいトークンに更新する。  
3. GitHub に誤コミットしていた場合は、そのコミットを修正し、履歴からも消す方法を検討する（必要なら GitHub の案内を参照）。
